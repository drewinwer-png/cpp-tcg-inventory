# TCG Inventory System 🃏

A robust, console-based CRUD (Create, Read, Update, Delete) application built in modern C++. This project demonstrates object-oriented programming principles and data management by simulating an inventory system for Trading Card Game (TCG) collectors.

## 🚀 Key Features
* **Interactive UI:** A continuous command-line interface for seamless user interaction.
* **Object-Oriented Design:** Strong encapsulation and class-based architecture (`Card` and `InventoryManager` modules).
* **Dynamic Storage:** Utilizes C++ Standard Template Library (`std::vector`) for efficient memory management.
* **Persistent Data:** Features an export module that saves the current database to a standard `.csv` file, making the data fully compatible with Excel and external databases.

## 🛠 Tech Stack
* **Language:** C++11 / Modern C++
* **Concepts:** OOP, STL Containers, File I/O (`<fstream>`), Input Validation.
